# HTML & CSS 

  

## Overview 

HTML structures web content, while CSS styles it. 

  

## Key Features 

- Core web technologies 

- Declarative syntax 

- Foundation for all web development 

  

## Basic Example 

```html 

<!DOCTYPE html> 

<html> 

  <body> 

    <h1>Hello, Web!</h1> 

  </body> 

</html> 

 ```
